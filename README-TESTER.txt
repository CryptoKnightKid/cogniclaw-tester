CogniClaw Tester Package

This package was prepared for external testing.

Included:
- cogniclaw source files
- project docs/config templates that are safe to share

Excluded:
- .env files
- secrets/tokens/keys
- memory/backups
- caches/build artifacts
- local databases
- node_modules/venvs

Before running:
1. Review setup docs inside cogniclaw/
2. Create your own .env/config from templates if needed
3. Install dependencies locally
